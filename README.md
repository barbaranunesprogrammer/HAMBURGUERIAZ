# 🍔 HamburgueriaZ

Aplicativo Android desenvolvido como prática de Desenvolvimento Mobile.  
O app simula uma hamburgueria onde o cliente pode montar seu pedido e enviá-lo por e-mail.

---

## 📱 Funcionalidades

- Inserir nome do cliente
- Escolher adicionais:
  - Bacon 🥓
  - Queijo 🧀
  - Onion Rings 🧅
- Selecionar quantidade de hambúrgueres
- Cálculo automático do preço total
- Exibição do resumo do pedido
- Envio do pedido por e-mail 📧

---

## 🛠️ Tecnologias utilizadas

- Java
- Android Studio
- XML (Layout)
- Intents (envio de e-mail)

---

## 📂 Estrutura do Projeto

- `MainActivity.java` → Lógica do app
- `activity_main.xml` → Interface do usuário
